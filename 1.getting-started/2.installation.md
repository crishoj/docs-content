# Installation

## Prerequisites

- Vue 3 or Nuxt 3.
- Vite (strongly recommended for automatic optimizations)
- Node.js: `14.18.0`, `16.12.0`, or higher.
- Terminal: To run `npm`/`yarn`/`pnpm` commands.
